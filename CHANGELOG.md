# AWS SSO Profiles Changelog

## [Initial Version] - {PR_MERGE_DATE}

- Browse AWS SSO accounts grouped by account ID
- Open AWS Management Console for any account and role
- Multi-session support: select up to 5 accounts and open them simultaneously
- Region picker with 17 built-in AWS regions
- Stage detection with color-coded icons (production, staging, dev, etc.)
- Usage tracking with optional count badges
- Copy actions for Account ID, Profile Name, Role Name, and export command
- Smart account display names derived from profile names
- Role-based icons and colors (Admin, ReadOnly, PowerUser, Developer)
